## Usage

### Register in config

Register in your config file (e.q. config.neon).

```neon
services:
	formatter.money: 
		class: Minetro\Formatter\NumberFormatter('Kc')
	formatter.weight: 
		class: Minetro\Formatter\NumberFormatter('kg', '~')
		setup:
			- setThousand(NULL)
			- setDecimal(0)
	nette.latteFactory:
		setup:
			- addFilter(money, [@formatter.number,format])
			# or
			- addFilter(money, @formatter.number::format)
```
